# Step 3: Create Document Entity

```java
@Entity
@Table(name = "documents")
@Getter
@Setter
@NoArgsConstructor
@AllArgsConstructor
@Builder
public class Document {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    private String fileName;

    private String originalFileName;

    private String filePath;

    private String contentType;

    private Long fileSize;

    @Enumerated(EnumType.STRING)
    private DocumentType documentType;

    @ManyToOne
    @JoinColumn(name = "patient_id")
    private Patient patient;
}
```

# Step 7: Multipart Upload Endpoint

```java
@PostMapping("/upload")
public ResponseEntity<String> upload(
        @RequestParam("file") MultipartFile file,
        @RequestParam("patientId") Long patientId,
        @RequestParam("documentType")
        DocumentType documentType
) {

    documentService.upload(
            file,
            patientId,
            documentType
    );

    return ResponseEntity.ok("Uploaded");
}
```

# Step 11: Generate Unique Filename

Never save:

```text
insurance.pdf
```

because another patient can upload:

```text
insurance.pdf
```

and overwrite it.

Use UUID.

```java
String uniqueFileName =
        UUID.randomUUID()
        + "_"
        + file.getOriginalFilename();
```

Example:

```text
9f3c8a2d-insurance.pdf
```

# Step 13: Save File Physically

```java
Path filePath =
        uploadPath.resolve(
                uniqueFileName
        );

Files.copy(
        file.getInputStream(),
        filePath,
        StandardCopyOption.REPLACE_EXISTING
);
```

Now file exists on disk.

```text
uploads/
    9f3c8a2d-insurance.pdf
```

# Complete Upload Method

```java
@Transactional
public void upload(
        MultipartFile file,
        Long patientId,
        DocumentType documentType
) throws IOException {

    validate(file);

    String uniqueFileName =
            UUID.randomUUID()
            + "_"
            + file.getOriginalFilename();

    Path uploadPath =
            Paths.get(uploadDir);

    if(!Files.exists(uploadPath)) {

        Files.createDirectories(
                uploadPath
        );
    }

    Path filePath =
            uploadPath.resolve(
                    uniqueFileName
            );

    Files.copy(
            file.getInputStream(),
            filePath
    );

    Patient patient =
            patientRepository
                    .findById(patientId)
                    .orElseThrow();

    Document document =
            Document.builder()
                    .fileName(uniqueFileName)
                    .originalFileName(
                            file.getOriginalFilename()
                    )
                    .filePath(
                            filePath.toString()
                    )
                    .contentType(
                            file.getContentType()
                    )
                    .fileSize(
                            file.getSize()
                    )
                    .documentType(
                            documentType
                    )
                    .patient(patient)
                    .build();

    documentRepository.save(document);
}
```

# Step 16: Download Endpoint

Clinician wants to view uploaded report.

```java
@GetMapping("/{id}")
public ResponseEntity<Resource>
download(
        @PathVariable Long id
)
```

Find document.

```java
Document document =
        documentRepository
        .findById(id)
        .orElseThrow();
```

Load file:

```java
Resource resource =
        new UrlResource(
                Paths.get(
                        document.getFilePath()
                ).toUri()
        );
```

Return:

```java
return ResponseEntity.ok()
        .header(
                HttpHeaders.CONTENT_DISPOSITION,
                "attachment; filename="
                + document.getOriginalFileName()
        )
        .body(resource);
```
