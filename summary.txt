# Step 7: Create AI Service

```java
@Service
@RequiredArgsConstructor
public class IntakeAiService {

    private final ChatClient chatClient;

    public String summarizeMedicalHistory(
            String medicalHistory
    ) {

        String prompt = """
                You are a medical intake assistant.

                Summarize the patient intake.

                Rules:
                - No diagnosis
                - No treatment advice
                - No assumptions

                Return:

                Main Concern:
                Symptoms:
                Medical History:
                Medications:
                Allergies:

                Patient Input:

                %s
                """
                .formatted(medicalHistory);

        return chatClient.prompt()
                .user(prompt)
                .call()
                .content();
    }
}
```

# Step 9: Service Layer

This is where everything happens.

```java
@Service
@RequiredArgsConstructor
public class IntakeService {

    private final IntakeRepository intakeRepository;

    private final IntakeAiService intakeAiService;

    public IntakeResponse submit(
            IntakeRequest request
    ) {

        IntakeForm form =
                intakeMapper.toEntity(request);

        intakeRepository.save(form);

        String summary =
                intakeAiService
                        .summarizeMedicalHistory(
                                request.getMedicalHistory()
                        );

        form.setAiSummary(summary);

        intakeRepository.save(form);

        return intakeMapper.toResponse(form);
    }
}
```

# What Actually Happens Here?

Patient enters:

```text
I have had headaches for 2 weeks.

I have diabetes since 2018.

Taking Metformin daily.

No allergies.
```

This text is sent to OpenAI.

OpenAI returns:

```text
Main Concern:
- Headaches for 2 weeks

Symptoms:
- Persistent headaches

Medical History:
- Diabetes (since 2018)

Medications:
- Metformin

Allergies:
- None reported
```

Then:

```java
form.setAiSummary(summary);
```

stores it.

DB:

| id | medical_history | ai_summary    |
| -- | --------------- | ------------- |
| 1  | Long paragraph  | Short summary |

# Production Improvement

Don't make patient wait.

Current flow:

```text
Save
 ↓
Call OpenAI
 ↓
Wait 3 sec
 ↓
Response
```

Bad UX.

Instead:

```text
Save Intake
 ↓
Return Success Immediately
 ↓
Background Thread
 ↓
Call OpenAI
 ↓
Store Summary
```

Use:

```java
@EnableAsync
```

and

```java
@Async
```

This is what many real systems do.
